# 🚀 Space Sentials

**Space Sentials** is a single-player space shooter game built entirely from scratch in **C++** with **OpenGL**.  
No external libraries were used — every object, enemy, and environment was manually drawn using points, lines, and shapes.  

Prepare yourself for an interstellar battle across **three challenging levels**, where precision and reflexes are the keys to survival.

---

## 🎮 How to Play

1. **Download & Unzip**  
   - Extract the contents of the `.rar` file.

2. **Run the Game**  
   - Launch the game by double-clicking the `SpaceSentials.exe` file.

3. **Controls**  
   - Arrow keys: Move your spaceship  
   - Spacebar: Fire lasers  
   - Esc: Exit game  

---

## 🌌 Features

- **Handcrafted Graphics**: Every star, ship, and asteroid drawn manually with OpenGL primitives.  
- **Three Levels of Action**: Increasing difficulty with unique enemy patterns.  
- **Pure C++ & OpenGL**: No external libraries — just raw code and creativity.  
- **Immersive Gameplay**: Fast-paced shooting with responsive controls.  

---

## 🛠️ Tech Stack

- **Language**: C++  
- **Graphics**: OpenGL  
- **Design Philosophy**: Manual rendering using points, lines, and shapes — no shortcuts, no libraries.
